# NBA DraftKings Lineup Optimizer Frontend

A simple, clean web interface to display the optimal DraftKings NBA lineup generated by the lineup-optimizer Lambda function.

## Features

- Clean, modern UI with gradient design
- Displays all 8 lineup positions (PG, SG, SF, PF, C, G, F, UTIL)
- Shows key stats: Total Salary, Remaining Budget, Projected Fantasy Points
- Responsive design that works on desktop and mobile
- Refresh button to reload the latest lineup
- Mock data mode for testing without API

## Setup

### Option 1: View with Mock Data (Testing)

1. Open `index.html` in your browser
2. The page will display sample lineup data
3. This is useful for testing the UI without connecting to the real Lambda function

### Option 2: Connect to Real Lambda API

1. Set up an API Gateway endpoint for your `lineup-optimizer` Lambda function
2. Open `index.html` in a text editor
3. Find the configuration section at the top of the `<script>` tag:
   ```javascript
   const API_ENDPOINT = 'YOUR_LAMBDA_API_GATEWAY_URL_HERE';
   const USE_MOCK_DATA = true;
   ```
4. Replace `YOUR_LAMBDA_API_GATEWAY_URL_HERE` with your actual API Gateway URL
5. Change `USE_MOCK_DATA` to `false`
6. Save the file and open in your browser

### Option 3: Serve with Python (Recommended for development)

```bash
cd frontend
python -m http.server 8000
```

Then open http://localhost:8000 in your browser.

## API Gateway Setup (AWS)

To connect this frontend to your Lambda function:

1. Go to AWS API Gateway console
2. Create a new REST API or HTTP API
3. Create a GET endpoint that triggers your `lineup-optimizer` Lambda function
4. Enable CORS if hosting the frontend on a different domain
5. Deploy the API and copy the endpoint URL
6. Update the `API_ENDPOINT` in `index.html`

### Example Lambda Response Format

The Lambda function should return:

```json
{
  "statusCode": 200,
  "body": {
    "success": true,
    "lineup_size": 8,
    "total_salary": 48500,
    "remaining_salary": 1500,
    "total_predicted_fp": 312.5,
    "players": [
      {
        "SLOT": "PG",
        "PLAYER": "Player Name",
        "POSITION": "PG/SG",
        "SALARY": 9000,
        "PREDICTED_FP": 45.5,
        "GAME_DATE": "2025-11-17"
      }
      // ... 7 more players
    ]
  }
}
```
